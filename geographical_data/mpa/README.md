# Marine Protected Areas

version: 2022-08-04

Marine protected areas (MPA) are protected areas of seas. MPAs restrict human activity for a conservation purpose, typically to protect natural or cultural resources.

Data extracted from the [CCAMLR GIS database](https://gis.ccamlr.org/) 2022-08-04 with the following links

- [Shape file EPSG:4326](https://gis.ccamlr.org/geoserver/gis/ows?service=WFS&version=1.0.0&request=GetFeature&typeName=gis:mpas&srsName=EPSG:4326&outputFormat=SHAPE-ZIP)
- [Shape file EPSG:6932](https://gis.ccamlr.org/geoserver/gis/ows?service=WFS&version=1.0.0&request=GetFeature&typeName=gis:mpas&srsName=EPSG:6932&outputFormat=SHAPE-ZIP)
- [Shape file ESRI:102020](https://gis.ccamlr.org/geoserver/gis/ows?service=WFS&version=1.0.0&request=GetFeature&typeName=gis:mpas&srsName=EPSG:102020&outputFormat=SHAPE-ZIP)
- [GeoJSON EPSG:6932](https://gis.ccamlr.org/geoserver/gis/ows?service=WFS&version=1.0.0&request=GetFeature&typeName=gis:mpas&srsName=EPSG:6932&outputFormat=application%2Fjson)
- [KML](https://gis.ccamlr.org/geoserver/gis/wms/kml?layers=gis:mpas)
